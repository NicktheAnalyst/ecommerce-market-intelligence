# E-Commerce Market Intelligence Platform

## Overview

A Python-based automation platform that monitors competitor e-commerce websites, collects market intelligence, enriches the data with public APIs, stores it in PostgreSQL, analyzes market trends, and visualizes the results in Power BI.

## Features

- Automated web scraping with Playwright
- Public API integration
- PostgreSQL database
- Market intelligence analysis
- Power BI dashboards
- Automated reporting
